# Gearhead's Gazette Image Standards

Use these standards for Gazette post images so the same asset works cleanly across article pages, homepage modules, and future preview surfaces.

## Core Standard
- Use a `16:9` image ratio
- Master image size: `1200 x 675 px`
- Save images in `assets/gazette/posts/`
- File names should match the article slug

## Example
- Article file: `content/lipo-batteries-101.md`
- Image file: `assets/gazette/posts/lipo-batteries-101.jpg`

## Usage Goal
Each Gazette image should be usable for:
- Article hero images
- Homepage Gearhead's Gazette module thumbnails
- Future preview cards or featured content blocks

## Composition Guidance
- Keep the main subject centered so responsive cropping stays safe
- Avoid text-heavy images because thumbnails may crop or scale down

## Preferred Formats
- `.jpg`
- `.webp`
