# Content Model

Each blog post should include:

## Core Elements
- Title
- Featured image
- Introduction paragraph
- Body content (sections with headings)
- Internal links (products, categories, related posts)
- Call to action (CTA)

## Optional Elements
- Bullet lists
- Embedded images
- Tips or callouts

## Structure Goals
- Easy to scan
- Consistent across all posts
- Flexible but not breakable

- ## Tone Reference

Use content/brushed-vs-brushless-motors.md as the tone benchmark.

All future posts should match this voice:
- conversational
- garage-style
- practical, not overly technical

The goal is to balance structure with flexibility.
